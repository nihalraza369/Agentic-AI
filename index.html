Problem Identification & Goal Setting
Definition: System ko samajhna hota hai ki kya problem hai aur kya achieve karna hai.
Kaam: Goal define karna.
Environment Sensing
Definition: System apne environment se data collect karta hai.
Kaam: Sensors, databases, ya user inputs se data lena.
Data Processing & Analysis
Definition: Collected data ko process aur analyze karna.
Kaam: Meaningful insights nikalna.
Decision Making
Definition: Analysis ke baad decisions lena.
Kaam: Goals ke hisab se actions plan karna.
Action Execution
Definition: Decisions ko execute karna.
Kaam: Actions lena, jaise robot movement ya email send karna.
Feedback & Learning
Definition: Actions ke results ko monitor karna.
Kaam: Feedback ke through decisions ko improve karna.
